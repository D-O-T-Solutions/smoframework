CRITICAL MISMATCHES
===================

1. current_target_ is IP:Port
   but get_session() expects node_id (64-char hex)
   Result: All file commands FAIL

2. cmd_connect() doesn't track peer node_id
   It only stores IP:Port
   Result: No way to find peer session after handshake

3. No method to find node_id by IP:Port
   RoutingTable only supports ID -> IP:Port lookup
   Result: Cannot map IP:Port back to node_id

4. Nodes command incomplete
   No method to iterate all nodes from RoutingTable
   Result: Shows empty table

5. No disconnect command
   Only exit (which terminates entire app)
   Result: Users cannot switch targets gracefully
